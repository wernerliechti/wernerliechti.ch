# Werner Liechti - Personal Website

A modern, responsive personal website for Werner Franz Liechti - Bitcoin Speaker, Unternehmer & Gründer.

## Features

- **Fully Responsive**: Mobile-first design that works on all devices
- **Smooth Scrolling**: Single-page navigation with smooth scroll behavior
- **SEO Optimized**: Complete meta tags, structured data (JSON-LD), and semantic HTML
- **Modern Design**: Clean, minimal design with orange (#f97316) accent color
- **Contact Form**: Integrated contact form with mailto functionality
- **Portfolio Sections**: Startups, Speaking, and Photography showcases

## Structure

```
├── index.html          # Main HTML file
├── style.css          # All CSS styles and responsive design
├── script.js          # JavaScript functionality
└── README.md          # This file
```

## Color Palette

- **Background**: Deep black (#0a0a0a) and dark gray (#1a1a1a)
- **Primary Accent**: Bold orange (#f97316)
- **Text**: White for headings, light gray (#ccc) for body text

## SEO Features

- **Meta Title**: "Werner Liechti | Bitcoin Speaker, Unternehmer & Gründer Schweiz"
- **Keywords**: Bitcoin Speaker Schweiz, Startup Speaker Schweiz, Unternehmer Schweiz
- **Structured Data**: Person, SpeakableSpecification, and FAQ schemas
- **Semantic HTML**: Proper use of header, main, section, article, and footer elements

## Sections

1. **Hero**: Main introduction with CTA button
2. **Über mich**: Personal introduction
3. **Startups**: Portfolio of companies and projects
4. **Speaking**: Topics, references, FAQ, and contact form
5. **Fotografie**: Photography portfolio preview
6. **Footer**: Contact information and social links

## Technical Features

- **Progressive Enhancement**: Works without JavaScript, enhanced with JS
- **Accessibility**: ARIA labels, semantic markup, keyboard navigation
- **Performance**: Optimized CSS and minimal dependencies
- **Modern CSS**: Grid, Flexbox, CSS variables, and smooth animations

## Contact Form

The contact form uses a mailto: link approach to open the user's email client with pre-filled information. This ensures no backend is needed while still providing a professional contact experience.

## Deployment

This is a static website that can be deployed to any web server or CDN. Simply upload all files to your web root directory.
